# 地方政府债务现金流匹配优化模型
---

## 一.项目背景

截至2025年末，中国地方政府债务余额已超过**53万亿元**，平均利率约**2.86%**，平均剩余年限**10.5年**。如何有效管理未来现金流、降低融资成本，成为地方政府财政管理的核心课题。

本项目基于**现金流匹配（Dedication）**和**免疫策略（Immunization）**两种经典方法，利用线性规划构建最低成本的债券组合，以覆盖未来8年的地方债还本付息需求。通过真实市场数据验证，发现了**负溢价**等有趣现象，为地方政府债务管理提供了量化决策支持。

---

## 二.数据来源

### 1. 负债数据
- 基于某市未来8年债务偿还计划模拟（等额本金+利息），总负债现值**4864.69亿元**。
- 负债结构：每年6月/12月各支付一半，符合地方债付息规律。

### 2. 债券数据
- 来源：**中国地方政府债券公开数据**（CSV文件，包含135只债券，筛选后29只可用）。
- 关键字段：`债券名称`、`发行日期`、`到期日期`、`发行规模(亿元)`、`票面利率(%)`。
- 筛选条件：剩余年限≤8年且未到期，最终获得29只代表性债券。

### 3. 国债收益率曲线
- 采用中国债券信息网最新数据（2026年3月），期限包括1,2,3,5,7,10,30年。
- 线性插值得到第1-8年各年收益率（**1.27% - 1.75%**）。

## 三.方法论与技术栈

### 技术栈
- **Python 3.8+**：核心编程语言
- **Pandas/NumPy**：数据处理与数值计算
- **SciPy.optimize.linprog**：线性规划求解（HiGHS求解器）
- **Matplotlib**：可视化分析
- **Jupyter Notebook**：交互式开发与展示

### 模型方法

#### 1. 现金流匹配（Dedication）
- 目标：最小化组合购买成本，使得未来每年债券产生的现金流（利息+本金）恰好等于当年负债需求。
- 约束：每只债券购买数量≥0，期初盈余=0，允许各年盈余结转。
- 线性规划形式：
  ```
  Min Σ P_i * x_i
  s.t. Σ (CF_{i,t} * x_i) + z_{t-1} - z_t = L_t,  ∀t
       x_i ≥ 0, z_t ≥ 0, z_0 = 0
  ```

#### 2. 免疫策略（Immunization）
- 目标：最小化成本，同时匹配负债的**现值、久期、凸性**，从而对冲利率平行移动风险。
- 久期/凸性计算公式：
  ```
  D = Σ [t * PV(CF_t)] / PV
  C = Σ [t*(t+1) * PV(CF_t)] / [PV * (1+r)^2]
  ```
- 线性规划约束：
  ```
  Σ P_i * x_i = PV_负债
  Σ [(D_i - D_负债) * P_i * x_i] = 0
  Σ [(C_i - C_负债) * P_i * x_i] = 0
  ```

#### 3. 敏感性分析
- 从影子价格推导隐含利率期限结构，与国债曲线对比。
- 计算利率平行移动对组合价值的影响（±100bps）。

#### 4. 风险度量指标
- 期限分布：加权平均到期年限、剩余年限标准差
- 集中度：HHI指数、最大单券占比、前三大集中度
- 流动性评分：基于发行规模、剩余年限的复合指标
- 匹配度：各年现金流覆盖率

## 四.关键发现

### 1. 负溢价现象（Dedication）
- Dedication最低成本：**4584.58亿元**
- 负债总现值：**4864.69亿元**
- **成本溢价：-280.11亿元（-5.76%）**

> 在当前市场利率低于票面利率的环境下，债券价格普遍低于面值（89-96元），通过大量购买折价债券，Dedication策略实现了低于负债现值的成本，相当于为地方政府节省了5.76%的融资成本。

### 2. 免疫策略完美复制负债特征
- Immunization成本：**4864.69亿元**（等于负债现值）
- 久期误差：**0.00**
- 凸性误差：**0.00**

> 仅用3只债券的组合即可精确匹配负债的现值、久期和凸性，证明中国债券市场存在足够丰富的工具来复制任意负债特征。

### 3. 策略对比
| 策略 | 成本(亿元) | 溢价率 | 持有债券数 | 加权久期 | 最大集中度 |
|------|------------|--------|------------|----------|------------|
| Dedication | 4584.58 | -5.76% | 5 | 5.23年 | 25.7% |
| Immunization | 4864.69 | 0.00% | 3 | 4.31年 | 44.2% |

### 4. 敏感性分析
- 影子价格隐含的收益率曲线与国债曲线基本吻合，验证了模型定价的合理性。
- 利率平行移动±100bps时，Dedication组合价值变化约±2.3%，风险暴露可控。

## 五.实际意义

1. **对地方政府**：
   - 在当前低利率环境下，通过提前置换或预融资，可节省大量财政支出（本案例中节省280亿元）。
   - Dedication策略提供绝对安全的现金流匹配，适合风险厌恶型财政管理。

2. **对投资机构**：
   - 折价债券存在套利机会，可构建低成本组合满足特定负债需求。
   - 免疫策略可快速复制负债，用于资产负债管理（ALM）。

3. **对政策制定者**：
   - 负溢价现象反映了市场对利率下行的预期，为货币政策提供参考。
   - 债券市场深度足以支持复杂的风险管理需求。
